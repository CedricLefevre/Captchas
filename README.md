# Captchas
In class, we learnt how to develop a simple captcha (with 5 letters or numbers).

How to run it ?

- Download UwAmp
- Run UwAmp
- Open UwAmp's files
- Put my files in a repertory (name it as you want)
- Put the reportory in UwAmp's files
- Open the navigator with UwAmp
- Run index.php

You ran it !
